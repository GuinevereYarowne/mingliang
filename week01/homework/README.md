关于第一题的查找素数并打印
我的思路如下：
通过命令行传入两个参数：开始值和结束值
计算这两个值之间的所有素数
怎么找呢？
对于一个数 n，检查从 2 到√n 之间是否有能整除 n 的数，当然得先看看这个数是否大于2
将结果写入文件，格式要求：
文件名格式：mingliang_primes_start_end.txt
内容格式：素数以半角空格隔开，最后一个素数后没有空格
程序最后需要打印
整体计算时间，查了一下，有time这个函数，可以记录
找到的素数的个数,这个也好记录
需要编写单元测试，这个，查了一下，是对编写的实现功能的函数进行测试，放一些数，看看这些函数的功能是否完善

关于第二题的计算简单计算题
思路如下：
因为格式还有一定要求，不留空格空行，但读取时要考虑输入文件的一些东西，所以使用bufio.Scanner读取文件，它能自动处理 Windows 的\r\n和 Linux 的\n换行符，返回干净的行内容（不含\r或\n）
然后还需要找运算符，在算式中找到第一个有效运算符（+、-、*、/），然后，可能有的数是负数，需要跳过
然后还需要把运算符左右数字分别提取为浮点数
